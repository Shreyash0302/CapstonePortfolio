    Capstone Project


    Title

    Developing an integrated system using swarm intelligence for optimization of disaster management tools


    Objective

  • Literature survey on current use of swarm intelligence models and their applications.

  • Identifying models of swarm intelligence and parameters from research papers

  • To develop An intelligent automated system or model which will be used to automatically combine models and parameters


    • Problem Statement

    Swarm intelligence is a collective behavior exhibited by decentralized, self-organized systems consisting of numerous interacting individuals or agents that work together to accomplish complex tasks. There are several swarm intelligence models that have been developed to solve various problems. But the problem with them is that in those models the basic learner is selected manually, the combination of each basic learner is attempted until the optimal combination is found. Hence, the process was not very efficient so in future the entire process can be made more automated and intelligent. An intelligent optimization algorithm can be used to simultaneously select the base learner and optimize the parameters of the selected basic learner. Looking at the above statement I have defined a problem statement that aims to build an integrated system using swarm intelligence to combine the models and parameters automatically.
